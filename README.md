# AI-InterviewPro

AI-InterviewPro is a Spring Boot and React-based mock interview platform that analyzes uploaded resumes, extracts text using Apache Tika, and generates personalized AI interview questions. Users can answer via text or voice and receive structured feedback, all without storing personal data.
